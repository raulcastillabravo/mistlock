This guide shows you how to work with **AWS S3** locally using **MinIO** and **Python**. This lab demonstrates how to integrate S3 libraries (**Boto3**, **PyArrow**, and **Deltalake**) for local data pipelines and object storage emulation.

## Prerequisites

- [Docker](https://www.docker.com/get-started) installed and running.
- [Dev Containers extension](vscode:extension/ms-vscode-remote.remote-containers) installed (optional).

## How to execute

<Steps>

1. **Setup Environment**:
   <Tabs>
      <TabItem label="Dev Container (recommended)">
         Open **VS Code** in the project folder and execute this command in the **Command Palette**:
         ```bash
         > Dev Containers: Reopen in Container
         ```
      </TabItem>
      <TabItem label="Manually">
         1. **Start Infrastructure**: Launch the required containers.
            ```bash
            docker compose up -d
            ```
         2. **Run the setup** script to install tools and dependencies.
            ```bash
            scripts/setup.sh
            ```
      </TabItem>
   </Tabs>

2. **Run the Example**:
   ```bash
   python main.py
   ```

</Steps>

## How to debug

1. **main.py**:
   - **Open**: Open `main.py`.
   - **Breakpoints**: Set breakpoints in the code.
   - **Run**: Press `F5` to start debugging.

2. **Tests**:
   - **Open**: Open a test file (e.g., `tests/components/test_s3_boto.py`).
   - **Breakpoints**: Set breakpoints in the test code.
   - **Run**: Use the VS Code **Testing** tab and click the **Debug Test** icon next to the test you want to debug.

## How to test

- **Individually**: Run tests individually from the VS Code **Testing** tab.
- **All tests**: Run the automated script:
  ```bash
  scripts/run_tests.sh
  ```

## Validate results

<Tabs>
   <TabItem label="MinIO GUI">
      - **Open**:
        ```bash
        http://localhost:9001
        ```
      - **Credentials**: Use the `MINIO_ROOT_USER` and `MINIO_ROOT_PASSWORD` defined in your `.env`.
      - **Verify**: Check the `bronze` and `silver` buckets to see the uploaded files.
   </TabItem>
   <TabItem label="AWS CLI">
      - **Run**: List the contents of the bronze bucket:
        <Tabs syncKey="cli-runner">
           <TabItem label="Global">
              ```bash
              aws s3 ls s3://bronze --recursive --profile minio
              ```
           </TabItem>
           <TabItem label="Mise">
              ```bash
              mise exec -- aws s3 ls s3://bronze --recursive --profile minio
              ```
           </TabItem>
        </Tabs>
   </TabItem>
   <TabItem label="MinIO CLI">
      - **Enter Shell**:
        ```bash
        scripts/minio_cli.sh
        ```
      - **Verify**:
        ```bash
        mc ls data/bronze/ --recursive
        ```
   </TabItem>
</Tabs>

## Clean Up

```bash
docker compose down -v
```
