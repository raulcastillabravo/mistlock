Este repositorio enseña cómo desarrollar para la Nube de forma gratuita, sin necesidad de crear una cuenta ni de introducir una tarjeta de crédito, emulando servicios de AWS, Azure y Google Cloud de forma local.

## Características Clave

* **Gratuito y sin cuentas**: Todas las tecnologías son gratuitas y no requieren crear una cuenta en ningún sitio.
* **100% compatible**: El código que desarrollas localmente es 100% compatible con la Nube real.
* **Autocontenido**: Cada ejemplo es independiente e incluye todo lo necesario para ejecutarse.
* **Listo para ejecutar**: Los ejemplos están listos para ejecutarse sin necesidad de realizar cambios en el código.
* **Dockerizado**: Todos disponen de entornos de desarrollo dockerizados.
* **Interfaces gráficas**: Uso de herramientas gráficas para gestionar el entorno local de la Nube.
* **Bien documentado**: Todos los ejemplos están documentados tanto en inglés como en español.

## Inicio Rápido

Para ejecutar cualquiera de los ejemplos o proyectos:

1. **Prerrequisitos**:
   * Tener instalado y en ejecución [Docker](https://www.docker.com/get-started).
   * (Opcional pero recomendado) Instalar [VS Code](https://code.visualstudio.com/) y la [extensión Dev Containers](vscode:extension/ms-vscode-remote.remote-containers).
2. **Abrir un ejemplo**: Abre la carpeta del ejemplo deseado (por ejemplo, `src/aws/mves/s3-garage/`) en VS Code.
3. **Reabrir en Contenedor**: Abre la paleta de comandos (`F1` o `Ctrl/Cmd+Shift+P`) y selecciona **Dev Containers: Reopen in Container**.
4. **Ejecutar**: Una vez que el entorno termine de configurarse, sigue las instrucciones del README del ejemplo.
