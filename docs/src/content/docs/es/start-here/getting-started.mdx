**MistLock** es un catálogo de entornos de desarrollo Cloud que combina Docker, Dev Containers, emuladores y servicios Open Source para que puedas desarrollar en local como si estuvieses en la nube, pero **100% gratis, sin cuentas y sin tarjeta de crédito**.

Usa estos entornos para aprender Cloud, iniciar tu proyecto o incorporarlos a un proyecto existente. 

El objetivo es que entiendas **cómo desarrollar en local usando un servicio Cloud**; cuando lo consigas, la gran mayoría de las veces solo necesitarás la nube para desplegar.

## Prerrequisitos

- [Docker](https://www.docker.com/get-started) instalado y en ejecución.
- [VS Code](https://code.visualstudio.com/) con la [extensión Dev Containers](vscode:extension/ms-vscode-remote.remote-containers) instalada *(opcional pero recomendada)*.

## Labs: MVEs vs Projects

A cada ejemplo del catálogo lo llamamos **Lab**, y hay dos tipos de Labs: MVE (Minimal Viable Examples) y Proyectos:
- **MVE**: se centra en un **único servicio** Cloud: cómo emularlo y qué herramientas usar.
- **Proyectos**: combina **varios servicios** Cloud en el mismo entorno local para mostrar cómo se integran en un flujo real.

## Cómo empezar

1. **Selecciona** un Lab del catálogo o navegando por el sidebar.

2. **Descarga** el código usando el botón de descarga que hay en cada Lab o clonando el repositorio de GitHub.

3. **Abre** el Lab en tu IDE.

4. **Ejecuta** el Lab. Cada Lab tiene sus propias instrucciones, pero generalmente es así:

   <Tabs>
      <TabItem label="Dev Container">
         Abre VS Code en la carpeta del Lab y abre la **paleta de comandos** con  `Ctrl+Shift+P`:
         ```bash
         Dev Containers: Reopen in Container
         ```
         Ejecuta el Lab:
         ```bash
         python main.py
         ```
      </TabItem>
      <TabItem label="Sin Dev Container">
         Inicia la infraestructura:
         ```bash
         docker compose up -d
         ```
         Configura el entorno:
         ```bash
         ./scripts/setup.sh
         ```
         Ejecuta el Lab:
         ```bash
         python main.py
         ```
      </TabItem>
   </Tabs>
