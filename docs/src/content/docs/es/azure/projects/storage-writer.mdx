Esta guía te muestra cómo trabajar con **Azure Functions** localmente usando **Azurite**, **Docker Compose** y **Python**. Este lab demuestra cómo crear una función con trigger HTTP que sube archivos al almacenamiento blob de Azurite.

## Prerrequisitos

- [Docker](https://www.docker.com/get-started) instalado y en ejecución.
- [Dev Containers extension](vscode:extension/ms-vscode-remote.remote-containers) instalada (opcional).

## Cómo ejecutar

<Steps>

1. **Configurar el entorno**:
   <Tabs>
      <TabItem label="Dev Container (recomendado)">
         Abre **VS Code** en la carpeta del proyecto y ejecuta este comando en la **Paleta de Comandos**:
         ```bash
         > Dev Containers: Reopen in Container
         ```
      </TabItem>
      <TabItem label="Manual">
         1. **Instala Azure Functions Core Tools**: Sigue la [guía de instalación oficial](https://learn.microsoft.com/en-us/azure/azure-functions/functions-run-local).
         2. **Instala las dependencias**:
            ```bash
            pip3 install uv && uv sync
            ```
      </TabItem>
   </Tabs>

2. **Iniciar la infraestructura**: Lanza el contenedor de Azurite:
   ```bash
   docker compose up -d
   ```

3. **Iniciar el host de funciones**:
   ```bash
   func start --prefix ./upload-function/
   ```

4. **Ejecutar el ejemplo**:
   ```bash
   python main.py
   ```

</Steps>

## Validar resultados

**Verifica con la [extensión Azure Storage](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azurestorage)**:
- **Abre**: Abre la vista **Azure** en VS Code (icono de Azure en la barra lateral).
- **Conecta**: En **Storage Accounts**, haz clic en **Attach Storage Account** > **Attach to a local emulator**.
- **Navega**: Usa la configuración predeterminada de Azurite para explorar visualmente tus contenedores y blobs.

## Limpieza

```bash
docker compose down -v
```

## Solución de problemas

| Problema | Solución |
|----------|----------|
| Conexión rechazada | Asegúrate de que Azurite está en ejecución: `docker ps` |
| Módulo no encontrado | Reinstala las dependencias de la función: `cd upload-function && pip install -r requirements.txt` |
