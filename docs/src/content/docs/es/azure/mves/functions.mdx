Esta guía te muestra cómo trabajar con **Azure Functions** usando **Python** y **Azure Functions Core Tools**. Este lab demuestra cómo desarrollar, ejecutar, depurar, testear y validar una función con trigger HTTP localmente.

## Prerrequisitos

- [Docker](https://www.docker.com/get-started) instalado y en ejecución.
- [Dev Containers extension](vscode:extension/ms-vscode-remote.remote-containers) instalada (opcional).

## Cómo ejecutar

<Steps>

1. **Configurar el entorno**:
   <Tabs>
      <TabItem label="Dev Container (recomendado)">
         Abre **VS Code** en la carpeta del proyecto y ejecuta este comando en la **Paleta de Comandos**:
         ```bash
         > Dev Containers: Reopen in Container
         ```
      </TabItem>
      <TabItem label="Manual">
         1. **Ejecuta el script** de configuración para instalar herramientas y dependencias.
            ```bash
            scripts/setup.sh
            ```
         2. **Iniciar la infraestructura**: Lanza los contenedores necesarios.
            ```bash
            docker compose up -d
            ```
      </TabItem>
   </Tabs>

2. **Iniciar el host de funciones**:
   <Tabs>
      <TabItem label="Terminal">
         ```bash
         func start
         ```
      </TabItem>
      <TabItem label="Run and Debug">
         - **Abre**: Abre la pestaña **Run and Debug** en VS Code.
         - **Ejecuta**: Selecciona **Attach to Python Functions** y pulsa `F5`. El emulador se inicia y el depurador se conecta automáticamente.
      </TabItem>
      <TabItem label="Azure Functions Extension">
         - **Abre**: Abre la barra lateral de **Azure** y navega a **Workspace → Local Project → Functions**.
         - **Ejecuta**: Haz clic en el botón **Start debugging** para iniciar la aplicación de funciones.
      </TabItem>
   </Tabs>

3. **Ejecutar el ejemplo**:
   <Tabs>
      <TabItem label="Python">
         ```bash
         python main.py
         ```
      </TabItem>
      <TabItem label="cURL">
         - **Admin** (éxito):
           ```bash
           curl "http://localhost:7071/api/get_secret?username=admin"
           ```
         - **Otro usuario** (prohibido):
           ```bash
           curl "http://localhost:7071/api/get_secret?username=guest"
           ```
      </TabItem>
      <TabItem label="REST Client">
         - **Instala**: [REST Client](vscode:extension/humao.rest-client) desde el Marketplace de VS Code.
         - **Abre**: Abre `http/get_secret.http` en VS Code.
         - **Ejecuta**: Haz clic en **Send Request** encima de cualquier bloque de petición.
      </TabItem>
   </Tabs>

</Steps>

## Cómo depurar

1. **function_app.py**:
   - **Abre**: Abre `function_app.py`.
   - **Breakpoints**: Añade breakpoints dentro de `get_secret`.
   - **Ejecuta**: Desde la pestaña **Run and Debug**, selecciona **Attach to Python Functions** y pulsa `F5`.

2. **main.py**:
   - **Abre**: Abre `main.py`.
   - **Breakpoints**: Añade breakpoints en la función `main`.
   - **Ejecuta**: Pulsa `F5` para iniciar la depuración.

3. **Tests**:
   - **Abre**: Abre un archivo de test (p. ej., `tests/test_functions.py`).
   - **Breakpoints**: Añade breakpoints en el código de test.
   - **Ejecuta**: Usa la pestaña **Testing** de VS Code y haz clic en el icono **Debug Test** junto al test que quieras depurar.

## Cómo testear

- **Individualmente**: Ejecuta tests individualmente desde la pestaña **Testing** de VS Code.
- **Todos los tests**: Ejecuta el script automatizado:
  ```bash
  scripts/run_tests.sh
  ```

## Validar resultados

<Tabs>
   <TabItem label="Terminal">
      - **Ejecuta**: Ejecuta el script principal y revisa su salida:
        ```bash
        python main.py
        ```
      - **Verifica**: El usuario definido en `ADMIN_USERNAME` (`.env`) debería recibir una respuesta `200` con el valor del secreto. Cualquier otro usuario debería recibir `403`.
   </TabItem>
   <TabItem label="Logs">
      - **Abre**: Revisa el terminal donde se está ejecutando `func start`.
      - **Verifica**: Cada petición registra una línea como `Python HTTP trigger function processed a request.`
   </TabItem>
</Tabs>

## Limpieza

```bash
docker compose down -v
```
