## Labs: MVEs y Proyectos

A cada ejemplo del catálogo lo llamamos **Lab**. Los Labs son independientes y autocontenidos: incluyen todo lo necesario para ejecutarse sin modificar el código. Existen dos tipos:

<CardGrid>
  <Card title="MVE (Minimal Viable Example)">
    Se centra en un único servicio Cloud: cómo emularlo y qué herramientas usar. Puede haber más de un MVE por servicio si hay varias opciones de emulación disponibles.
  </Card>
  <Card title="Proyecto">
    Combina varios servicios Cloud en el mismo entorno local para mostrar cómo se integran entre sí en un flujo real.
  </Card>
</CardGrid>

## AWS (Amazon Web Services)

<CardGrid>
  <Card title="⚡ Lambda">
    Emula AWS Lambda con **SAM CLI**. Desarrolla y prueba funciones serverless localmente.
  </Card>
  <Card title="🪣 S3">
    Emula AWS S3 con **Garage** o **MinIO**. Almacenamiento de objetos compatible con la API de S3.
  </Card>
  <Card title="🐘 RDS (PostgreSQL)">
    Emula Amazon RDS con una instancia local de **PostgreSQL**.
  </Card>
  <Card title="🔴 ElastiCache">
    Emula Amazon ElastiCache con un servidor local de **Redis**.
  </Card>
  <Card title="🔄 Step Functions">
    Emula AWS Step Functions con **LocalStack** para orquestar flujos de trabajo serverless.
  </Card>
</CardGrid>

## Microsoft Azure

<CardGrid>
  <Card title="⚡ Azure Functions">
    Emula Azure Functions localmente con las herramientas oficiales de desarrollo de Azure.
  </Card>
  <Card title="🪣 Blob Storage">
    Emula Azure Blob Storage con **Azurite**, el emulador oficial de almacenamiento de Microsoft.
  </Card>
  <Card title="🔬 Databricks">
    Emula un entorno de **Databricks** localmente para procesamiento y análisis de datos.
  </Card>
  <Card title="🔴 Cache for Redis">
    Emula Azure Cache for Redis con un servidor local de **Redis**.
  </Card>
</CardGrid>

## Google Cloud Platform

<CardGrid>
  <Card title="🔥 Cloud Functions">
    Emula Google Cloud Functions con **Firebase Emulator Suite**. Desarrolla funciones serverless sin conexión.
  </Card>
  <Card title="🐘 Cloud SQL">
    Emula Google Cloud SQL con una instancia local de **PostgreSQL**.
  </Card>
  <Card title="🔴 Memorystore">
    Emula Google Memorystore con un servidor local de **Redis**.
  </Card>
</CardGrid>
