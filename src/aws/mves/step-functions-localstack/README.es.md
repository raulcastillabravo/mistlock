# Step Functions (LocalStack)

Esta guía te muestra cómo trabajar con AWS Step Functions localmente usando LocalStack, Python y el AWS Toolkit para VS Code. Este lab demuestra un flujo de incorporación de usuarios con ejecución paralela de Lambda y creación de usuarios IAM.

> 📖 La documentación completa de este lab está disponible en:
