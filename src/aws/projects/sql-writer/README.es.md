# SQL Writer

Esta guía te muestra cómo construir un escenario de nube híbrida usando LocalStack y una instancia externa de PostgreSQL. Este lab demuestra cómo una AWS Lambda recupera secretos de Secrets Manager e interactúa con una base de datos fuera del entorno AWS.
