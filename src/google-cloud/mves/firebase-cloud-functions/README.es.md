# Firebase Cloud Functions

Esta guía te muestra cómo trabajar con Google Cloud Firebase Functions localmente usando el Firebase Emulator Suite. Este lab demuestra una función HTTPS síncrona que recupera un secreto de Secret Manager basándose en el nombre de usuario.

> 📖 La documentación completa de este lab está disponible en:
> https://mistlock.dev/google-cloud/mves/firebase-cloud-functions/
