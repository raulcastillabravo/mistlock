# Databricks

This guide shows you how to simulate a Databricks environment locally using Docker, MinIO as S3-compatible storage, and PostgreSQL as a persistent Hive Metastore. This lab demonstrates how to develop and test Spark and Delta Lake ETLs locally using Databricks Runtime 15.4 LTS.

> 📖 Full documentation for this lab is available at:
> https://mistlock.dev/azure/mves/databricks/
